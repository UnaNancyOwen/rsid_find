rsid_find
=========

This is tool for find all connected RealSense ID devices.  

Sample
------
You can use this tool with following command.  
### Command
```
rsid_find.exe
```
### Output
```
Find RealSense ID Devices
-------------------------
* number of devices : 1
* deivice 1
  - serial type : USB
  - port        : \\.\COM3
```

Environment
-----------
* Windows 10
* Visual Studio 2019 (C++)
* CMake 3.17 (or later)
* RealSense ID SDK v0.13.0 (or later)
